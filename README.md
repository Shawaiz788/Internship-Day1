# Internship Day 1 - Git & GitHub Workflow Practice

This repo is a cloned copy of an open-source project used to practice core Git and GitHub collaboration workflows during the first day of internship.

> **Note:** The project itself was taken from a public GitHub repository. It is not my original work - it was used purely as a sandbox for Git practice.

---

## Purpose

The goal was to get hands-on experience with a real-world Git workflow by working inside an existing codebase, covering:

- Cloning a remote repository
- Creating and switching branches
- Making commits with meaningful messages
- Opening pull requests
- Reviewing and merging branches

---

## Workflow Practiced

```bash
# Clone the repo
git clone https://github.com/Shawaiz788/Internship-Day1.git

# Create a new branch
git checkout -b my-branch

# Make changes, then stage and commit
git add .
git commit -m "your commit message"

# Push the branch to remote
git push origin my-branch

# Then open a Pull Request on GitHub and merge into main
```
